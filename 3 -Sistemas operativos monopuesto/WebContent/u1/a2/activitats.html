Activitats

Instal·lació i configuració global de VirtualBox

L’objectiu d’aquesta activitat és aprendre a instal·lar i configurar VirtualBox.

Instal·leu al vostre ordinador l’última versió de VirtualBox, recordeu que heu de descarregar la versió que coincideixi amb el vostre sistema operatiu i equip informàtic: Un cop instal·lat VirtualBox, configureu les següents opcions:

  1. La ruta a la carpeta on voleu guardar per defecte els arxius de les màquines virtuals.
  2. Les actualitzacions automàtiques perquè VirtualBox comprovi cada setmana si hi ha actualitzacions disponibles.
  3. Canvieu la llengua de VirtualBox al català.
  4. Instal·leu el paquet d’extensió “Oracle VM VirtualBox Extension Pack”.

Per instal·lar el programa VirtualBox cal que anem a la seva pàgina web oficial (www.virtualbox.org) i descarreguem el programa corresponen pel nostre sistema operatiu. Per exemple, si el nostre sistema operatiu és Windows 10, caldrà que seleccionem l’opció Windows hosts. Si el nostre sistema operatiu és Ubuntu 20.04, seleccionem l’opció Linux distributions per després seleccionar el programari per la nostra versió d’Ubuntu, en aquest cas la 20.04. Un cop descarregat el programa, l’instal·lem al nostre equip seguint els passos de l’assistent d’instal·lació.

  1. Per configurar la carpeta per defecte on volem guardar els arxius de les màquines virtuals cal anar a Fitxer > Preferències > General, caldrà seleccionar una carpeta amb suficient espai per guardar tots els arxius de les màquines virtuals.
  2. Per configurar les actualitzacions automàtiques anem a Fitxer > Preferències > Actualització i a l’apartat 1 vegada per: seleccionem 1 setmana.
  3. Per canviar l’idioma de VirtualBox anem a Fitxer > Preferències > Llengua i seleccionem Català.
  4. Per instal·lar el paquet d’extensió “Oracle VM VirtualBox Extension Pack” cal que anem a la pàgina (web oficial de VirtualBox) i descarreguem el paquet. El paquet té extensió “.vbox-extpack”. Un cop tenim el paquet emmagatzemat al nostre equip anem a Fitxer > Preferències > Extensions, cliquem la icona Afegeix un paquet nou i indiquem la ruta on es troba emmagatzemat el paquet.

Creació d'una màquina virtual en un sistema operatiu amfitrió utilitzant VirtualBox

L’objectiu d’aquesta activitat és aprendre a crear màquines virtuals utilitzant VirtualBox.

Utilitzeu el programari VirtualBox per crear una màquina virtual per instal·lar el sistema operatiu Windows 10 Home. Per fer-ho seguiu els següents passos:

  1. Comproveu quins són els requisits de maquinari (memòria RAM i espai en disc dur) necessaris per instal·lar Windows 10 Home (64 bits).
  2. Utilitzeu l’assistent de creació de màquines virtuals per especificar el nom de la màquina virtual, el tipus i versió del sistema operatiu convidat.
  3. Seleccioneu la quantitat de memòria RAM que es reservarà per la màquina virtual.
  4. Afegiu un nou disc a la màquina virtual, amb format natiu VDI i ubicat de forma dinàmica.

  1. Per conèixer els requeriments mínims de maquinari de Windows 10 Home, cal buscar aquesta informació a la pàgina web oficial de Microsoft. En aquest cas, aconsellen 2 GB per la memòria RAM i 32 GB d’espai en disc dur disponible.
  2. A la primera finestra de l’assistent de creació de màquines virtuals especifiquem el nom de la màquina virtual, la carpeta on volem emmagatzemar els arxius necessaris per a la creació de la màquina virtual, el tipus de sistema operatiu que volem instal·lar (en aquest cas “Microsoft Windows”) i la seva versió (en aquest cas “Windows 10 64-bit”).
  3. A la segona finestra de l’assistent de creació de màquines virtuals, seleccionem la quantitat de memòria RAM. Indiquem 2048 MB.
  4. A la tercera finestra de l’assistent de creació de màquines virtuals seleccionem l’opció Crea un disc virtual ara. Aquesta selecció ens porta a un nou assistent de creació de discos durs. En aquesta primera finestra seleccionem el tipus de fitxer que volem crear per al disc dur virtual nou, en aquest cas VDI. Al següent pas seleccionem que el fitxer pugui créixer a mesura que s’utilitza, per això marquem Ubicat de forma dinàmica. Per últim seleccionem el nom del disc dur virtual nou, la seva ubicació i la seva mida, en aquest cas 32 GB.

Configuració d'una màquina virtual creada amb VirtualBox

L’objectiu d’aquesta activitat és aprendre a configurar màquines virtuals creades amb VirtualBox.

Utilitzeu la màquina virtual creada a la activitat 2 (“Creació d’una màquina virtual en un sistema operatiu amfitrió utilitzant VirtualBox”) per fer les següents configuracions:

  1. Compartiu el porta-retalls entre la màquina virtual i la màquina amfitrió.
  2. Permeteu que es puguin arrossegar i deixar anar elements directament entre el sistema operatiu convidat i el sistema operatiu amfitrió.
  3. Introduïu una descripció a la màquina virtual on indiqueu que la màquina virtual es correspon a la màquina virtual creada a l’activitat 2.
  4. Afegiu un nou monitor a la màquina virtual, en total tindreu dos monitors en aquesta màquina virtual.
  5. Compartiu una carpeta ubicada a la màquina amfitrió amb la màquina virtual.

Per fer totes les configuracions demanades a l’activitat cal seleccionar la màquina virtual que volem configurar i clicar l’opció Paràmetres que podem trobar al clicar amb el botó dret del ratolí sobre de la màquina virtual:

  1. Per compartir el porta-retalls cal seleccionar l’opció Bidireccional a l’opció General > Avançat > Porta-retalls compartit. Aquesta opció permet compartir el porta-retalls en els dos sentits.
  2. Per poder arrossegar i deixar anar elements cal seleccionar l’opció Bidireccional a l’opció General > Avançat > Arrossega i deixar anar. Aquesta opció permet arrossegar i deixar anar els elements en els dos sentits.
  3. Per introduir una descripció a la màquina virtual cal anar a General > Descripció i escriure el text desitjat al quadre de text.
  4. Per afegir un nou monitor a la màquina virtual cal anar a Pantalla > Pantalla > Nombre de monitors i seleccionar el número 2.
  5. Per compartir una carpeta de la màquina amfitrió amb la màquina virtual cal anar a la secció Carpetes compartides i seleccionar la icona Afegeix una compartició a la dreta de la finestra. Això farà aparèixer una nova finestra on hem d’especificar la ruta de la carpeta i triar un nom compartit que la màquina virtual utilitzarà per accedir a la carpeta compartida.

Utilització d'una màquina virtual creada amb VirtualBox

L’objectiu d’aquesta activitat és aprendre a utilitzar màquines virtuals.

Utilitzeu la màquina virtual creada a l’activitat 2 (Creació d’una màquina virtual en un sistema operatiu amfitrió utilitzant VirtualBox). Tot i que la màquina virtual no té cap sistema operatiu convidat instal·lat podeu fer el que es demana als següents apartats:

  1. Exporteu la màquina virtual a un arxiu de tipus OVA. Comproveu que l’arxiu s’ha creat a la ruta seleccionada.
  2. Feu una instantània per guardar l’estat de la màquina actual i comproveu que s’ha creat un nou arxiu VDI a la carpeta on s’emmagatzemen els arxius de la màquina virtual.
  3. Feu una clonació completa de la màquina virtual, comproveu que apareix una nova màquina a la llista de màquines virtuals a la finestra de gestió de VirtualBox.
  4. Importeu l’arxiu OVA que heu creat al primer apartat, comproveu que apareix una nova màquina a la llista de màquines virtuals a la finestra de gestió de VirtualBox.
  5. Per últim elimineu la màquina clonada a l’apartat 3 d’aquesta activitat.

  1. Per exportar una màquina virtual a un arxiu cal seleccionar la màquina que volem exportar i seleccionar Fitxer > Exporta una aplicació virtual a la barra de menú. Al quadre de diàleg que apareix, només cal seleccionar la màquina virtual que volem exportar. Al següent pas seleccionem Open Virtualization Format i indicarem el nom de la màquina exportada amb extensió OVA i la ubicació on s’emmagatzemarà l’arxiu exportat. Com opció addicional podem incloure un arxiu de manifest per guardar la configuració original de la màquina. Per últim polsem el botó Exporta per començar el procés d’exportació. Amb aquest format d’exportació de màquines virtuals només es crea un arxiu, podem veure l’arxiu a la ubicació seleccionada per guardar els arxius de les màquines virtuals exportades.
  2. Per fer una instantània, fem clic al nom de la màquina virtual, després seleccionem la icona Llista que està al costat del nom de la màquina i seleccionem Instantànies. Després cliquem la icona Fes. Al crear una instantània caldrà guardar-la amb un nom que ens servirà com a referència. Per ajudar-nos a recordar l’estat de la instantània podem utilitzar el nom “Màquina inicial”. Per comprovar que s’ha creat un nou arxiu VDI, anem a la carpeta on es guarden els arxius de la màquina virtual i dins trobarem una nova carpeta anomenada Snapshots. Dins d’aquesta carpeta trobarem tants arxius VDI com instantànies de la màquina virtual fem.
  3. Per fer una clonació, cal clicar sobre la màquina virtual al tauler esquerre de la finestra de gestió de VirtualBox amb el botó dret i seleccionar l’opció Clona. Un cop hem seleccionat aquesta opció, omplim el nom i la ubicació de la màquina virtual clonada. Al següent pas triarem fer una clonació completa per copiar tots els arxius de la màquina virtual a un nou directori. Cliquem el botó Clona, a continuació es farà la clonació de la màquina virtual i podrem trobar la nova màquina al tauler de l’esquerra de la finestra de gestió de VirtualBox amb el nom que li hem assignat.
  4. Per importar un arxiu en format OVA cal seleccionar Fitxer > Importar una aplicació virtual a la finestra de gestió de VirtualBox. Al quadre de diàleg que apareix, seleccionem la font d’on importar la màquina virtual que serà l’arxiu creat al primer apartat. Després comprovem la configuració de la màquina virtual i per últim fem clic al botó Importa per importar la màquina virtual.
  5. Per eliminar la màquina virtual cal seleccionar-la a la finestra de gestió de VirtualBox i amb el botó dret del ratolí, seleccionar l’opció Eliminar. El quadre de diàleg de confirmació ens permetrà seleccionar si volem eliminar només la màquina virtual o també els arxius associats amb la màquina virtual. En aquest cas, ho eliminem tot.

Utilització de dispositius virtuals

L’objectiu d’aquesta activitat és aprendre a utilitzar màquines virtuals.

Per fer aquesta activitat utilitzeu la màquina virtual de l’activitat 2 (Creació d’una màquina virtual en un sistema operatiu amfitrió utilitzant VirtualBox). Feu les activitats que s’indiquen als següents apartats:

  1. Mostreu els fitxers de la màquina virtual a l’explorador d’arxius.
  2. Amb la màquina aturada, copieu el fitxer que conté el disc dur de la màquina virtual a una nova carpeta.
  3. Creeu una nova màquina virtual sense disc dur.
  4. Mostreu la configuració d’emmagatzematge de la màquina que heu creat i afegiu el disc dur que heu copiat de la màquina virtual creada al punt anterior. Mostreu novament la configuració d’emmagatzematge de la màquina virtual.
  5. Connecteu un dispositiu USB al sistema amfitrió. Obriu la configuració de la màquina virtual, localitzeu el dispositiu USB i apliqueu-lo a la màquina convidada.
  6. Inicieu la màquina convidada i observeu que li passa al dispositiu USB connectat a la màquina amfitriona.

  1. En primer lloc cal localitzar els fitxers que representen la màquina virtual al disc dur de la màquina amfitriona, per comprovar la seva ubicació ho fem a Fitxer > Preferències > General > Carpeta de la màquina per defecte. De forma predeterminada els trobem a “C:\Users\nom usuari\VirtualBox VMs\Nom maquina”. Ho podem comprovar accedint a la carpeta amb l’explorador d’arxius.
  2. Per copiar el fitxer del disc dur, cal localitzar els fitxers amb extensió .vdi a la ubicació anterior, per després copiar-lo a una altra ubicació del disc dur.
  3. En crear una nova màquina virtual, en el tercer pas, Disc dur, cal seleccionar la primera opció: No afegeixis un disc dur virtual. Al crear la màquina virtual d’aquesta forma apareixerà una advertència que, sense disc dur, no es podrà instal·lar cap sistema operatiu, però sí que es podrà iniciar la màquina per xarxa o amb una imatge .ISO o disc òptic virtual (tal com succeeix a un equip real).
  4. Amb la màquina sense disc dur creada, seleccionem Paràmetres i dintre d’aquests, a l’esquerra seleccionem Emmagatzematge. Es pot observar que el controlador SATA no té cap dispositiu connectat. Cal fer clic en la icona de dispositiu amb el símbol “+“ que està a sota de la llista de dispositius d’emmagatzematge titulada Afegeix una nova adjunció d’emmagatzematge i aquí podem seleccionar unitat òptica o disc dur. Per l’activitat cal seleccionar Disc Dur. En aquest punt apareix una finestra on es poden seleccionar tots els discs durs que VirtualBox té registrats. En aquest cas, com que hem creat un nou disc dur al copiar i enganxar, aquest nou fitxer no està registrat al sistema, per tant cal seleccionar l’opció Afegeix i seleccionar el fitxer que havíem copiat. Ara el disc dur ja està configurat correctament a la màquina virtual.
  5. Per configurar els dispositius USB cal clicar a la màquina virtual amb el botó dret del ratolí i seleccionar Paràmetres > USB. Com que no hi ha cap filtre de dispositiu creat, cal crear un fent clic a la icona que hi ha a la dreta anomenada Afegeix un filtre USB nou amb tots els camps amb valor igual al dispositiu USB connectat a l’ordinador amfitrió. D’aquesta forma VirtualBox detecta els diferents dispositius USB que estan connectats a la màquina amfitriona i tan sols cal seleccionar el dispositiu USB que ens interessa connectar a la màquina convidada. Una vegada seleccionat apareixerà en la configuració USB de la màquina virtual.
  6. A l’iniciar la màquina convidada amb el filtre USB activat, el dispositiu USB es desconnectarà a la màquina amfitriona (ja no estarà disponible), ja que VirtualBox ha estat configurat perquè quan detecti el dispositiu el connecti a la màquina convidada.

Utilització de VBoxManage i Vagrant per la gestió de màquines virtuals

L’objectiu d’aquesta activitat és crear una màquina virtual utilitzant una caixa de Vagrant i comprovar la seva configuració utilitzant tant l’entorn gràfic de VirtualBox com la seva eina VBoxManage.

Seguiu els següents passos per crear, comprovar la configuració i eliminar una màquina creada a partir d’una caixa de Vagrant:

  1. Creeu una màquina virtual utilitzant la caixa de Vagrant anomenada “bento/ubuntu-20.04”.
  2. Visualitzeu la configuració de la màquina que heu creat utilitzant la interfície gràfica de VirtualBox.
  3. Visualitzeu de nou la configuració de la màquina però aquesta vegada utilitzeu l’eina VBoxManage de Virtual Box. Utilitzeu l’ordre VBoxManage showvminfo.
  4. Elimineu la màquina virtual creada i la caixa de Vagrant del sistema operatiu amfitrió.

  1. Per crear una màquina virtual utilitzant una caixa de Vagrant, en aquest cas la caixa “bento/ubuntu-20.04”, cal crear un directori al nostre sistema operatiu amfitrió amb el nom de la màquina virtual. Per fer-ho, si estem utilitzant Windows, anem a l’explorador d’arxius i creem una nova carpeta anomenada, per exemple, “Ubuntu”. Obrim l’intèrpret d’ordres que desitgem utilitzar i ens ubiquem a la carpeta creada. Un cop ens trobem a la carpeta, executem l’ordre vagrant init bento/ubuntu-20.04 i després l’ordre vagrant up. Al cap d’un moment, podrem veure la màquina virtual creada a la finestra de gestió de VirtualBox.
  2. Per visualitzar la configuració de la màquina que hem creat utilitzant la interfície gràfica de VirtualBox cliquem amb el botó dret sobre la màquina virtual i seleccionem l’opció Paràmetres. En aquesta finestra podem observar la configuració predeterminada d’aquesta màquina virtual, característiques del processador, memòria RAM, dispositius d’emmagatzematge, etc. Podeu aprofitar que heu obert la finestra de configuració per canviar el nom de la màquina a un nom més adequat, com per exemple, “Ubuntu 20.04”.
  3. Per utilitzar les ordres de VBoxManage tornem a obrir l’intèrpret d’ordres. Per mostrar la configuració de la màquina fem servir l’ordre VBoxManage showvminfo Ubuntu 20.04. En aquest cas podem veure moltes més opcions que les que hem pogut consultar a la interfície gràfica de VirtualBox.
  4. Per eliminar la màquina virtual utilitzem l’ordre vagrant destroy. Per eliminar la caixa Vagrant fem servir l’ordre vagrant box remove bento/ubuntu-20.04.

Creació i utilització d'un contenidor en un sistema operatiu Windows 10 utilitzant Docker Desktop

L’objectiu d’aquesta activitat és aprendre a crear i a utilitzar un contenidor utilitzant Docker Desktop.

Per crear un contenidor seguiu els següents passos:

  1. Instal·leu al vostre ordinador l’última versió de Docker Desktop, recordeu que heu de descarregar la versió que coincideixi amb el vostre sistema operatiu.
  2. Utilitzeu el programari Docker Desktop per crear un contenidor de Windows. Per fer-ho descarregueu la imatge “mcr.microsoft.com/windows/nanoserver:1809” de Docker Hub.
  3. Accediu a la interfície de línia d’ordres del contenidor i executeu l’ordre hostname. Aquesta ordre mostra quin és el nom de la màquina i el mostra per pantalla. Executeu la mateixa ordre a PowerShell del sistema operatiu amfitrió i comproveu que són dues màquines diferents, ja que no retornen el mateix nom d’equip.
  4. Atureu el contenidor per després eliminar-lo.

  1. El primer pas per instal·lar Docker Desktop es descarregar l’arxiu d’instal·lació de la pàgina web oficial de Docker. Si el nostre sistema operatiu és Windows 10 Pro, descarreguem l’arxiu d’instal·lació Windows Stable. Després de descarregar l’aplicació, cal executar l’arxiu d’instal·lació i seguir les instruccions fins que finalitzi l’assistent d’instal·lació, moment en el qual és necessari reiniciar el sistema.
  2. Per descarregar la imatge “mcr.microsoft.com/windows/nanoserver:1809” i ubicar-la al tauler de Docker utilitzem PowerShell de Windows. Un cop hem accedit a PowerShell escrivim l’ordre docker image pull mcr.microsoft.com/windows/nanoserver:1809 i comprovem que la imatge es troba disponible al tauler de Docker. Ara que ja tenim la imatge disponible al tauler de Docker per crear el contenidor cal executar-la. Per fer-ho seleccionem la imatge i seleccionem la icona RUN que apareix quan passem el ratolí per sobre la imatge.
  3. Per accedir al sistema operatiu del contenidor, anem a la vista Containers/Apps passem el ratolí per sobre del contenidor i cliquem a la icona CLI, un cop estem a la interfície de línia d’ordres escrivim l’ordre hostname i anotem el nom de l’equip. Posteriorment executem PowerShell del sistema operatiu Windows amfitrió, executem l’ordre hostname i comprovem que en aquest cas el nom de l’equip és un altre. Podeu observar a la figura l’execució de l’ordre hostname al sistema operatiu amfitrió i al sistema operatiu del contenidor.
  4. Per aturar el contenidor, des de la vista Containers/Apps passem el ratolí per sobre el contenidor i cliquem la icona Stop. Després cliquem sobre la icona Delete per eliminar-lo.

Documentació del procés general de creació i configuració d'una màquina virtual

L’objectiu d’aquesta activitat és documentar el procés de creació i configuració d’una màquina virtual.

Suposeu que teniu un ordinador amb el sistema operatiu Windows 10 i voleu crear una màquina virtual per instal·lar un sistema operatiu convidat Ubuntu 20.04.

Documenteu el procés de creació i configuració d’aquesta màquina virtual donant tots els detalls necessaris. Heu de donar les dades necessàries tant de la màquina real com de la màquina virtual.

Un possible exemple de documentació de creació i configuració de la màquina virtual proposada a l’enunciat és el següent:

Màquina real:

  1. Característiques tècniques de l’equip:
    • Microprocessador: Intel Core 2 Duo, 2.6 GHz
    • Targeta gràfica: Integrada
    • RAM: 8 GB
    • Disc dur: 500 GB
    • Targeta de xarxa: Marvell Yukon 88E8056 PCI-E Gigabit Ethernet Controller
  2. Sistema operatiu:
    • Windows 10 Home.
  3. Programes instal·lats:
    • Mozilla Firefox, versió 75.0, instal·lat: 29/04/2020, navegador web gratuït.
    • Avast, versió 20.2, instal·lat: 29/04/2020, programa antivirus gratuït.
    • LibreOffice, versió 6.4.3, instal·lat: 30/04/2020, paquet ofimàtica gratuït.
    • GIMP, versió 2.10.18 instal·lat: 30/04/2020, editor d’imatges gratuït.
  4. Configuració de xarxa:
    • Equip domèstic, connexió Wifi
    • IP: Obtenció automàtica
    • DNS: Obtenció automàtica
  5. Pedaços i actualitzacions instal·lats:
    • KB4549951, nova compilació del sistema operatiu, instal·lada el 16/04/2020.
    • Logitech - HIDClass - 1.10.86.0, actualització de controlador, instal·lada el 14/04/2020.
    • KB2267602, actualització de seguretat per Windows Defender Antivirus, instal·lada el 28/04/2020.
    • KB890830, eina d’eliminació de software malintencionat, instal·lada el 26/03/2020.

Màquina virtual:

  1. Característiques tècniques de l’equip:
    • Carpeta de màquines: C:\Users\IOC\VirtualBox VMs
    • Processador: 2 nuclis
    • Límit d’execució processador: 100%
