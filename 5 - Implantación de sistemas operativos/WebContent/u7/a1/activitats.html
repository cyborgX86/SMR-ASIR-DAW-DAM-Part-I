Activitats

Objectiu La finalitat d’aquesta pràctica consisteix en instal·lar en un servidor OpenLDAP i phpLDAPadmin per dur a terme una gestió senzilla dels usuaris i els grups d’LDAP.

Requeriments Un servidor Linux, per exemple podeu fer-ho amb una màquina virtual d’Ubuntu Server 10.04 LTS.

Enunciat

Cal que instal·leu el daemon de servidor d’LDAP, les utilitats d’LDAP i l’aplicació phpLDAPadmin.

- Primer de tot, instal·lem el daemon del servidor OpenLDAP slapd i el paquet ldap-utils, paquet que conté les utilitats d’administració d’LDAP:

sudo apt-get install slapd ldap-utils

- Un cop tenim el servidor amb la capacitat de gestió de domini LDAP, per tal de facilitar-nos l’administració instal·lem el paquet phpLDAPadmin:

sudo apt-get install phpldapadmin

Per tal que aquesta aplicació funcioni requereix tot un conjunt d’aplicacions (com per exemple el servidor web apache i l’intèrpret de php) que, en cas de no estar instal·lades, el sistema de gestió de paquets apt hauria de proposar-ne la instal·lació perquè phpLDAPadmin pugui funcionar.

Hi ha diferents opcions per tal de fer la configuració inicial de phpldapadmin. Podeu utilitzar el guió inclos a l’apache (/etc/phpldapadmin/apache.conf), o bé podeu copiar /usr/share/phpldapadmin a la vostra carpeta web arrel, situació que permetria fins i tot tenir instal·lacions múltiples de phpldapadmin.

La configuració més senzilla passa per editar el fitxer /etc/phpldapadmin/config.php. Probablement haureu d’editar les següents línies:

Línies del fitxer de configuració que cal editar
$ldapservers→SetValue($i,'server’,'name’,'El meu servidor LDAP’); // El nom que es mostra
$ldapservers→SetValue($i,'server’,'host’,'127.0.0.1’); // Adreça del servidor LDAP. És l’adreça local perquè el servidor web apache i el servidor LDAP són la mateixa màquina.
$ldapservers→SetValue($i,'server’,'port’,'389’); // Número de port
$ldapservers→SetValue($i,'server’,'base’,array(‘dc=example,dc=com’)); // dn de base
$ldapservers→SetValue($i,'login’,'string’,'uid=<username>,ou=People,dc=example,dc=com’);

Amb aquestes dades de configuració el phpldapadmin ja es podrà iniciar tot entrant l’adreça http://127.0.0.1/phpldapadmin o bé http://localhost/phpldapadmin des d’un navegador web del mateix servidor.
